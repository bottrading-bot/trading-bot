Notes:

- Put all downloaded files directly into this folder.
- Do not set `PREFERRED_BACKGROUND_FILENAME` if you want real rotation.
- Keep `PREFERRED_BACKGROUND_KEYWORD` empty for mixed clips, or set it to
  `minecraft`, `neon`, `keyboard`, `city`, etc. if you want a theme.
