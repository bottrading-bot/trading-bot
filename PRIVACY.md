# Privacy Policy for VideoBot

VideoBot is a local desktop automation tool that helps the creator generate short vertical videos and upload them to TikTok as drafts.

## Data We Access

When the creator connects their TikTok account, VideoBot may request basic account information through TikTok Login Kit, such as the creator profile identifier and display information required by TikTok.

## Data We Use

VideoBot uses TikTok authorization only to upload generated videos to the creator's TikTok draft inbox through the Content Posting API.

## Data Storage

VideoBot stores generated video files, captions, metadata, and local processing files on the creator's own computer. TikTok access tokens should be stored only as private environment variables and should not be shared publicly.

## Data Sharing

VideoBot does not sell user data. VideoBot only sends generated video files and related upload metadata to TikTok when the creator explicitly runs the upload flow.

## User Control

The creator can stop using VideoBot at any time and revoke app access in their TikTok account settings.

## Contact

For privacy questions, contact the app owner through the GitHub repository connected to this project.
